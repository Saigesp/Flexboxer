What is this?

Flexboxer is a small super-responsive LESS framework based on flexbox.

Why this exits?

Because I need a framework and I hate all others, especially Bootstrap.

Have I use it?

No, unless you work with me.

Install & use

To clone files to you local machine, run:

git clone https://github.com/Saigesp/Flexboxer.git

Compile files

To compile LESS files, run:

gulp less

To watch LESS files and compilie it when they change, run:

gulp watch

To build a minified CSS file, run:

gulp build
